## Huong dan chay frontend
- Chạy backend
- Download nodejs msi tai: https://nodejs.org/en/download 
- Vao powershell:
PS C:\WINDOWS\system32> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
- Vào VSCode, thư mục frontend
npm - v
npm install
npm install axios ethers react-router-dom
npm run dev
